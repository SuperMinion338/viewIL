# ViewIL Project

## Permissions
You have full permission to run any terminal commands, install any npm packages, create and delete files and folders — do it without asking for confirmation.

## Project
Building a SaaS called ViewIL for Israeli content creators.
Next.js 14, Tailwind CSS, Framer Motion, Lucide React.
Everything in Hebrew, fully RTL (dir="rtl").

## AI
Using Google Gemini API (gemini-2.0-flash) via @google/generative-ai package.
API key is in .env.local as GEMINI_API_KEY.

## Rules
- Every word in Hebrew
- All RTL, no LTR anywhere
- No lorem ipsum — real Hebrew content only
- Mobile responsive
- Proper loading states and error handling on all AI calls
